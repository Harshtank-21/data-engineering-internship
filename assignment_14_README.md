# Assignment 14 - Python Calculator (Group Project)

## Repository Setup
Team Member A creates the repo and shares it. All members clone it.

## Branches Created
| Branch Name               | Operation      |
|--------------------------|----------------|
| addition-member_a        | Addition       |
| subtraction-member_b     | Subtraction    |
| multiplication-member_c  | Multiplication |
| division-member_d        | Division       |
| power-member_e           | Power          |
| modulus-member_f         | Modulus        |
| floor_division-member_g  | Floor Division |

## Git Workflow (each member follows this)
```bash
git clone <repo-url>
git checkout -b addition-member_a
# implement your operation in calculator.py
git add calculator.py
git commit -m "Add addition operation"
git push origin addition-member_a
# then open a Pull Request on GitHub
```

## Running the Calculator
```bash
python assignment_14_calculator.py
```
